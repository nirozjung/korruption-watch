# Corruptions Nepal Backend
Corruption Watch Project. 
The goal of the project is to collect corruption data & dynamically generate Infographics based on the corruptions data.

## Backend 
- (NestJS + TypeORM + SQLite)

# Installation
- nppm install
- npm run start:dev
- API: GET http://localhost:4000


## Installation
npm install 
npm run start:dev

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
